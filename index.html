The Course

Below you will find an overview of what I have learnt so far on this course.

1 - Introduction to tech

In the first session, there was a general introduction to the world of tech which covered the following topics:

  • What a tech product is made of.
  • Overview of Programming languages and frameworks.
  • Frontend vs Backend programming.
  • "The cloud"
2 - Version controls

In the second session, there was a general introduction to the importance of version controls and how to implement this. We used Github and created our own profiles and learnt the uses of certain commands like push, pull and how to create and clone repositories.

We also learnt about branches and their impact on a project through a master branch and subsidiary branches.

3 - HTML Fundamentals

In session three the fundamentals of HTML were introduced to us whilst we were shown the implementation of this through the use of Visual Studio Code.

The following topics were covered:

  • Document structure.
  • Structural Tags.
  • Grouping and text tags.
4 - HTML Forms

In the fourth session we were introduced to HTML forms and how they are used. The importance of having the correct structure and an overview of the different input types along with element attributes were also covered.

5 - CSS Fundamentals

In the fifth session the fundamentals of CSS was introduced. CSS is the way in which you can correctly change the layout and display of your webpage.

The following topics were covered in this session:

  • Linking CSS to your HTML.
  • CSS Selectors.
  • Psuedo elements and classes.
  • Fonts
6 - CSS Box Model

In the sixth session the Box model was introduced. This is concept that all HTML elements can be considered as boxes. Each box has margins, borders, padding and content and this assists with conceptualising each elements position and the overall layout of the webpage.

The following topics were covered in this session:

  • What the box model is.
  • Different CSS positioning types
  • Everything is a box
  • Using chrome developer tools
7 - CSS Flexbox & Bootstrap

In the seventh session, we were introduced to the flexbox and the framework of bootstrap. This makes the construction and styling of responsive webpages easier and helps to solve issues with allignment etc.
